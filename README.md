# Data Science Projects

This repository contains a collection of data science projects focused on machine learning, predictive analytics, statistical analysis, and business problem solving.

## Projects

### Project 01 - Basic Data Cleaning Task
Load and clean a customer dataset by removing formatting issues, standardizing data types, and preparing it for analysis.

### Project 02 - Advanced Data Cleaning & Analysis Pipeline
A comprehensive python based data processing project demonstrating scalable, parameterized functions for cleaning, validating, and analyzing customer datasets using nested list structures and index based operations.

### Project 03 - Music Streaming Analysis
Comparative analysis of music streaming behavior between two cities. Tests the hypothesis that user activity differs by day of the week and varies between cities using real streaming data.
